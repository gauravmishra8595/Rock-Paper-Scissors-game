
# Rock-Paper-Scissors Game

## 🕹️ About

Rock-Paper-Scissors is a classic hand game played between two players where:

- **Rock** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Rock** 3

This version lets a user play against the computer with an engaging interface and dynamic score tracking. 4

---

## ✨ Features

- ✅ **Interactive User Interface**  
- 📊 **Score Tracking**  
- 🔄 **Responsive Design** for different screen sizes  
- 🎉 **Instant game result feedback**  
- 🖱️ Click or tap controls for easy gameplay 5

---

## 🧠 How to Play

1. Open the game in your browser.  
2. Click **Rock**, **Paper**, or **Scissors**.  
3. The computer will make a random selection.  
4. Compare moves and see who wins!  
5. Your score updates automatically each round. 6
